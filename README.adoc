:toc: left
:toclevels: 2

Satellite SSG implementation providing MCP-compatible interface to 28 static site generators.

== Overview

chicxulub--ssg is a satellite implementation in the hyperpolymath ecosystem that integrates with the link:https://github.com/hyperpolymath/poly-ssg-mcp[poly-ssg-mcp hub]. It provides adapters for 28 different static site generators via the Model Context Protocol (MCP).

=== Features

* **28 SSG Adapters** - Support for generators in 15+ programming languages
* **MCP Protocol** - Compatible with Model Context Protocol for AI integration
* **RSR Compliant** - Follows Rhodium Standard Repository guidelines
* **Security First** - CodeQL scanning, SHA-pinned actions, secrets detection
* **Deno Runtime** - Secure-by-default JavaScript/TypeScript execution

== Quick Start

[source,bash]
----
# Clone the repository
git clone https://github.com/hyperpolymath/chicxulub--ssg.git
cd chicxulub--ssg

# Setup development environment
just setup

# Run quality checks
just qa

# List available adapters
just list-adapters
----

== Development

=== Prerequisites

* link:https://deno.land[Deno] 1.x or later
* link:https://github.com/casey/just[Just] command runner
* Git

=== Commands

[source,bash]
----
just setup          # Initial setup
just build          # Check all adapters
just test           # Run tests
just lint           # Run linter
just fmt            # Format code
just qa             # Run all quality checks
just list-adapters  # List available adapters
just verify-adapter <name>  # Verify specific adapter
----

=== Git Hooks

Install development hooks:

[source,bash]
----
./hooks/install.sh
----

Configure in `hooks/config.sh` or create `hooks/config.local.sh` for personal overrides.
